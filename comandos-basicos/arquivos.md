# Manipulação de Arquivos

## mkdir

Cria uma pasta.

```bash
mkdir projetos
```

## touch

Cria um arquivo.

```bash
touch arquivo.txt
```

## cp

Copia arquivos.

```bash
cp origem.txt destino.txt
```

## mv

Move ou renomeia arquivos.

```bash
mv arquivo.txt novoarquivo.txt
```

## rm

Remove arquivos.

```bash
rm arquivo.txt
```
