# Exercício 02 - Organização de Projetos

## Objetivo

Aprender a criar e organizar diretórios utilizando comandos Linux.

## Estrutura Criada

```text
estudos/
├── linux/
├── git/
├── python/
└── cloud/
```

## Comandos Utilizados

```bash
mkdir estudos
cd estudos

mkdir linux
mkdir git
mkdir python
mkdir cloud

ls
```

## Explicação dos Comandos

### mkdir

Cria diretórios.

### cd

Permite navegar entre diretórios.

### ls

Lista arquivos e diretórios existentes.

## Aprendizados

- Como criar múltiplos diretórios.
- Como navegar entre pastas.
- Como organizar projetos de tecnologia.
- Importância da estruturação de arquivos.
